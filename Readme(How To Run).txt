
How to run the program:
1. If needed install JDK1.5(or JRE1.5) or later.
[ search internet by JDK1.5(or JRE1.5) in google(www.google.com)
and download the appropriate software and install ]
2. In windows: Double click the file
     Run.bat to run this program.
   In Linux: Possibly executing Run.sh will do(I haven't tested).

According to the jdk version, change the content of the
Compile.bat or Run.bat file if needed(by right clicking and selecting "Edit").
