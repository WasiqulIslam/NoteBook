# Note Book v1.1.1.0

This is an efficient and secure notebook or diary app that you can use in your PC.

Just download Notebook.exe and Settings.txt file to start using instantly. 
Modify password in the settings file as you like. 
Notebook.exe is a portable Windows application; no setup or installation is necessary.
Just run it.
